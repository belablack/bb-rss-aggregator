Fetches and publishes story posts from various RSS feeds.

== Description ==
Bela Black's RSS Aggregator is a WordPress plugin that fetches and publishes story posts from multiple RSS feeds automatically.

Features:
- Automate post creation from RSS feeds.
- Select post status (published or draft).
- Assign categories and set default categories.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "Settings > Bela's RSS Aggregator" to configure the options.

== Frequently Asked Questions ==

= How do I add RSS feeds? =
Go to the plugin settings under "Settings > Bela's RSS Aggregator" and add one feed URL per line.
